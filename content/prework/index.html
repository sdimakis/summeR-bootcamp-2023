Hello! Welcome to the 5th Annual UO R Bootcamp. There are a few things we’d like you to do before we get started to make things run as smoothly as possible.

If possible, please try to complete this checklist before the first UO R Bootcamp session. Thank you!


  1. Register for the UO R Bootcamp by completing this short registration survey. If you can’t make all of the days (or all of the sessions on a given day), that’s okay! We just ask that you register so that we can (1) send you the links to join the Zoom sessions and (2) get a better understanding of your familiarity with R. Registration is, of course, free.

  2. Make sure you have a Zoom account set up through the University of Oregon. We will use the same Zoom link for each day of the bootcamp. You will receive the link through email and slack.

  3. Join the Slack workspace. This is how we will primarily be communicating with each other, and it is the best place to ask questions outside of the scheduled bootcamp hours. We’ll be using the channel #bootcamp2022. You should have received an email invitation to join the SlackRs workspace. If you haven’t received an invitation, send Cameron an email at ckay@uoregon.edu. When you join, please introduce yourself and say hello in the #bootcamp2022 channel!

  4. Download and install both R and RStudio. R is free and, for the purposes of the bootcamp, you will only need the free version of RStudio. If you have any trouble downloading or opening R or RStudio, please let one of the UO R Bootcamp team members know, and we will do our best to help you solve the issue.

  5. Download the materials and exercises (these will be added by 09/16/2022) for the UO R Bootcamp. We will be working through the materials and exercises together during the bootcamp.

  6. Review the Code of Conduct.

  7. Optional - Download Google Chrome if you don’t have it already. The materials on this website are best viewed with Chrome. We’ve found that using other browsers (e.g., Firefox, Safari) can lead to some technical difficulties.

Looking forward to seeing you all soon. Hopefully we’ll have you loving R in no time!

-The UO R Bootcamp Team
